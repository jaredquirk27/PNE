# Persistent Narrative Engine (PNE)

## Purpose

PNE (Persistent Narrative Engine) is a long-term AI companion and narrative simulation platform.

The goal is not to create a chatbot.

The goal is to create persistent characters who:
- Remember past events
- Form relationships
- Maintain continuity
- Generate initiatives
- Progress stories
- Evolve over time
