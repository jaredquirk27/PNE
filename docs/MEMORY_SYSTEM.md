# Memory System

## Memory Pipeline

Conversation
↓
Memory Candidate
↓
Acceptance
↓
Canonization
↓
Knowledge Base
↓
Retrieval

## Categories

- Definition
- Preference
- Named Entity
- Identity
- Story Event
- Accomplishment

## Rules

Good memories:
- factual
- concise
- searchable
- entity-rich

Bad memories:
- Project discussion
- Transcript fragments
- 'I don't know'
- Generic relationship summaries
